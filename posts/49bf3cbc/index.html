Metronic 前端模板学习

作者:  KAKARROT    日期:  六月 05, 2019    分类:   前端

环境搭建

  1. 下载MAMP，MAMP是一个免费的本地服务器环境。
  2. 将模板文件夹放入C:\MAMP\htdocs路径下。
  3. 启动MAMP服务。
  4. 浏览器输入localhost，预览项目页面。

模板目录结构

路由设置

  1. '/'是根目录地址http://localhost/admin/public/
  2. 'demo/index.html'是首页地址http://localhost/admin/public/demo/index.html
  3. 'demo/login.html'是登录地址http://localhost/admin/public/demo/login.html

PHP文件拆分

  1. 将PHP文件内的DIV区域合理的拆分成若干个PHP文件。
  2. 再引入当前PHP文件内。
  3. 引入时，切记路径完整正确，可忽略文件名后缀。

@include('demo.layout.common.header.mobile')

工具类通用替换

  1. 将工具模块拆分成若干个PHP文件。
  2. 再引入当前PHP文件内。
  3. 引入时，切记路径完整正确，可忽略文件名后缀。

下拉框

@include('demo.components.filter.filter', ['key' => "类型", 'value' => ['支付宝','微信']])

搜索框

@include('demo.components.search.search', ['placeholder' => '订单号'])
