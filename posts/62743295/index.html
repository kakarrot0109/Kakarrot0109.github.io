MaxScript VScode 调试运行配置

作者: Kakarrot 日期: 五月 7, 2019  0:00:00 分类: MaxScript
  1. 安装vscode
  2. 安装扩展程序Language MaxScript
  3. 下载并编译MXSPyCOM.sln
  4. 将MXSPyCOM.exe放在 C 盘根目录
  5. 将initialize_COM_server.ms放在 3dsmax 各版本.\script\startup路径
  6. 打开命令面板，选择Configure Default Test Task
  1. 打开命令面板，选择Open Keyboard Shortcuts File
