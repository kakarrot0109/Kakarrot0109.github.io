IPython 变成了 Jupyter 庞大开源项目（一个交互和探索式计算的高效环境）中的一个组件。

安装方式

NumPy

NumPy（Numerical Python 的简称）是 Python 科学计算的基础包。

pip install numpy

pandas

pandas 提供了快速便捷处理结构化数据的大量数据结构和函数。

pip install pandas

matplotlib

matplotlib 是最流行的用于绘制图表和其它二维数据可视化的 Python 库。

pip install matplotlib

更新全部模块
