IKBC Poker 2 使用指南

作者:  KAKARROT    日期:  四月 22, 2020    分类:   随记

常见FAQ

Q: 左 WIN 键亮灯怎么关 ？

A: FN + Z

Q: 空格左灯怎么关 ？

A: FN + 右 Shift

Q: 空格右灯怎么关 ？

A: FN + 右 Ctrl

Q: 怎么打出 ~ ？

A: Shift + FN + ESC (注意顺序)

Q: 只能打出大写字母／大写锁定了

A: 再按一次 Shift，如果左 WIN 键亮灯，可以参照第一条关掉

Q: 关掉键盘的 LED 灯 ？（仅限于带背光版本）

A: FN + v

Q: w、a、s、d 打不出来，变成方向键了

A: FN + 空格键

Q: PN 键被锁定了，锁定编程层才能正常使用（FN + 右 Shift）才能正常用键码

A：键盘断电试试（亲测可行，可能是 bug）

LED 功能（适用于背光版本）

LED 功能键 功能 备注
FN V 全键 LED 亮（除左 WIN） 按压第一次

编程相关

编程相关 功能键 组合键 备注
FN R 复位键（慎重） 必须长按 12 秒
FN 右 Shift 切换至编程层 按压第一次（空格左灯亮）
FN 右 Shift 切换回编程层 按压第二次（空格左灯灭）
FN 右 Ctrl 进入编程模式 按压第一次（PN 灯亮）
FN 右 Ctrl 退出编程模式 按压第二次（PN 灯灭）
FN F 延时 15 ms 仅在编程模式下有效
FN G 延时 0.1 s 仅在编程模式下有效
FN H 延时 0.5 s 仅在编程模式下有效

编程举例

目标：把 a 编为 Ctrl + FN + a （mac 下切换屏幕）

  1. FN + 右 Ctrl 打开编程模式，空格右键亮起
  2. 点下 a 键， 接着按下 Ctrl + FN + a 组合键，放开，再点击 PN 键保存编辑
  3. FN + 右 Ctrl 退出编程模式，空格右灯熄灭
  4. PN + a => 屏幕切换，成功
  5. 现在我们锁定编程模式 FN + 右 Shift，按 a => 屏幕切换

特殊功能

FN 特殊功能 功能键 组合键 备注
FN Z 把 Shift 键变成按下一直发码，再按一下停止发码 按压第一次（左 WIN 灯亮）
FN Z Shift 键恢复正常 按压第二次（左 WIN 灯灭）
FN SPACE W，A，S，D 转换为 ↑←↓→ 按压第一次
FN SPACE ↑←↓→ 转换为 W，A，S，D 按压第二次

指拨开关说明

指拨开关说明 ON OFF 备注
开关 1 CAP = 左 WIN / 左 FN CAP 灯 = 左 WIN 灯 CAP = CAP CAP 灯 = CAP 灯 预设 = OFF
开关 1 左 WIN / 左 FN = CAP 左 WIN 灯 = CAP 灯 左 WIN / 左 FN = 左 WIN / 左 FN 左 WIN 灯 = 左 WIN 灯 预设 = OFF
开关 2 右 Ctrl = `~ 右 Ctrl = 右 Ctrl 预设 = OFF
开关 3 左 WIN = 左 FN 左 WIN = 左 WIN 预设 = OFF
开关 4 写保护键盘，按 FN + R 或当尝试编程按键都会导致 PN 灯快速闪烁一秒 然后熄灭并返回非编程模式的正常状态 预设 = OFF
