Typora+PicGo写作

作者:  Kakarrot    日期:  八月 6, 2020  0:00:00    分类:   随记

下载 PicGo

PicGo 下载地址

Typora 配置

进入 Typora 的偏好设置-图像进行如下设置：

  1. 插入图片时，选择”上传图片”
  2. 对本地、网络位置图片应用上述规则 勾选此 2 个选项
  3. 上传服务设定，选择 PicGo(app)并选择路径

SM.MS 获取 Token

登录sm.ms个人控制面板，在 API Token 生成”Secret Token”

PicGO 安装插件

安装插件之前，确保已经按照 Node.js

  1. 运行 PicGo-插件设置中搜索smms-user插件进行安装
  2. 运行 PicGo-图床设置中 SM.MS 登录用户中，Auth输入 Token 码

测试效果

  1. 重启 PicGo 软件
  2. 在 PicGo 中上传图片进行测试
  3. 在 Typora 中右键选择图片上传进行测试

SM.MS 有时会抽风，但是目前来说应该是免费中较好用的图床应用了。
